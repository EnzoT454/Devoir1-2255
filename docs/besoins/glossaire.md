## Termes Académiques

| Terme | Définition |
|-------|------------|
| Prérequis | Cours qu'un étudiant doit avoir réussi avant de s'inscrire à un autre cours. |
| Crédit | Unité de mesure évaluant la charge de travail d'un cours. |
| Cycle | Niveau universitaire d'un étudiant. |
| Temps plein/temps partiel | Statut de l'étudiant déterminé par le nombre de crédits inscrits. |
| Spécialisation | Concentration permettant d'approfondir un domaine spécifique. |
| Taux de réussite/échec | Proportion d'étudiants ayant réussi ou échoué un cours. |
| Conflit d'horaire | Chevauchement d'horaire empêchant de suivre deux cours simultanément. |
| Cheminement | Séquence de cours recommandée par programme pour chaque session. |
| Structure du programme | Répartition des crédits entre cours obligatoires, optionnels et hors programme. |
| Abandon | Désinscription d'un cours avant la date limite. |
| Inscription | Processus de sélection et d'inscription aux cours. |
| Charge de travail | Nombre d'heures estimé pour études, devoirs et évaluations. |
| Cours obligatoire | Cours requis pour l'obtention du diplôme. |
| Cours optionnel | Cours choisi parmi une liste limitée. |
| Session | Période académique (automne, hiver, été). |

## Termes Techniques et Discord

| Terme | Définition |
|-------|------------|
| Discord | Plateforme de communication utilisée pour recueillir les avis et expériences des étudiants concernant les cours du baccalauréat ou de la maîtrise. |
| Serveur Discord | Espace sur Discord qui regroupe les étudiants d'un programme où se déroulent les discussions. |
| Channel Discord | Sous-espace d'un serveur Discord dédié à un sujet précis comme un cours ou un club par exemple. |
| Plateforme d'entraide | Espace où les étudiants peuvent poser des questions, demander de l'aide sur les devoirs et partager des ressources concernant les cours. |
|utilisateur discord | Étudiant et membre du serveur qui participe aux discussions et partage ses avis sur les cours. |
| message discord | texte rédigée par un étudiant qui peut apparaître soit dans le serveur public ou bien dans une conversation privée entre étudiants |
| Pseudo discord | Nom visible d’un utilisateur sur Discord |
| Fil de discussion | une séquence de messages en lien avec un message spécifique |
