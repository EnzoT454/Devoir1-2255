## Termes Académiques

| Terme | Définition |
|-------|------------|
| Prérequis | Cours qu'un étudiant doit avoir réussi avant de s'inscrire à un autre cours. Sans ce cours préalable, l'inscription n'est pas autorisée par le système.|
| Crédit | Unité de mesure évaluant la charge de travail d'un cours. |
| Cycle | Niveau universitaire d'un étudiant (1er cycle = baccalauréat, 2e = maîtrise, 3e = doctorat). |
| Temps plein/temps partiel | Statut de l'étudiant déterminé par le nombre de crédits suivis durant la session. |
| Spécialisation | Concentration permettant d'approfondir un domaine spécifique dans le programme. |
| Taux de réussite/échec | Proportion d'étudiants ayant réussi ou échoué un cours. |
| Conflit d'horaire | Situation où deux cours ont des plages horaires qui se chevauchent, rendant impossible leur inscription simultanée. |
| Cheminement | Séquence de cours recommandée par programme pour chaque session, selon la progression des prérequis et les exigences du diplôme. |
| Structure du programme | Répartition des crédits entre cours obligatoires, optionnels et hors programme. |
| Abandon | Désinscription d'un cours avant la date limite. |
| Inscription | Processus de sélection et d'inscription aux cours pour une session donnée. |
| Charge de travail | Nombre d'heures estimé par cours pour les études, devoirs et évaluations. |
| Cours obligatoire | Cours requis pour l'obtention du diplôme. |
| Cours optionnel | Cours choisi parmi une liste proposée par le programme. |
| Session | Période académique officielle (automne, hiver, été). |

## Termes Techniques et Discord

| Terme | Définition |
|-------|------------|
| Discord | Plateforme de communication utilisée pour recueillir les avis et les expériences des étudiants concernant les cours du baccalauréat, de la maîtrise ou du doctorat. |
| Serveur Discord | Espace sur Discord qui regroupe les étudiants d'un programme où se déroulent les discussions. Chaque serveur contient plusieurs sous-canaux. |
| Channel Discord | Sous-espace d'un serveur Discord dédié à un sujet précis comme un cours ou un club par exemple. |
| Plateforme d'entraide | Espace où les étudiants peuvent poser des questions, demander de l'aide sur les devoirs et partager des ressources concernant les cours. |
|Utilisateur Discord | Étudiant et membre du serveur Discord qui participe aux discussions et partage ses avis sur les cours. |
| Message Discord | Texte rédigé par un étudiant qui peut apparaître soit dans le serveur public ou bien dans une conversation privée entre membres du serveur. |
| Pseudo discord | Nom visible d’un utilisateur sur Discord |
| Fil de discussion | une séquence de messages en lien avec un message spécifique à l'intérieur d'un canal discord. |
